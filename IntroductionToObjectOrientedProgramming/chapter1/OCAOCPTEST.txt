    /*Question 1
      Which of the following method signatures is a valid declaration of an entry point in a Java application?
        A. public void main(String[] args)
        B. public static void main()
        C. private static void start(String[] mydata)
        D. public static final void main(String[] mydata)

        Cevap:
           Java uygulamalarında bir giriş noktası, tek bir String[] argümanına, void dönüş türüne ve public ve static değiştiricilere sahip bir main() yönteminden oluşur.
           Girdi argümanındaki değişkenin adı önemli değildir.
           Seçenek A, static belirteci eksik, Seçenek B, String[] argümanını eksik, ve Seçenek C yanlış erişim değiştiricisi ve yöntem adına sahip.
           Yalnızca Seçenek D bu gereksinimleri karşılar.
           Not olarak, final değiştirici isteğe bağlıdır ve bir giriş noktası yöntemine eklenebilir.
    */

        C. It can be easily read and modified in a standard text editor.
